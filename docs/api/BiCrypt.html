BiCrypt

BiCrypt

A simple two-way encryption class.

This class is based on algorithms given in Applied Cryptography 2nd Ed.

If subclassed, the new encryption class must provide three methods:

  • encrypt_block(block)

  • decrypt_block(block)

  • block_size()
